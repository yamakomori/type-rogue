# ことばの小さな冒険 — MVP

子どものタイピング習慣をつくる、短時間のローグライク風タイピングゲームです。Astroのページ層に、ゲーム本体を単一のReact Islandとして載せています。現在はホームポジションから基本かなの入口（S00〜S09）までを対象にしたMVPです。

## 起動

```bash
npm run dev
```

ブラウザで表示されたローカルURLを開きます。進捗はブラウザのローカルストレージに保存されます。

## テスト

```bash
npm test
```

## MVPに含むもの

- 3問・約30〜60秒の短いプレイ
- S00〜S09 のホームポジション、上下段、基本かなカリキュラム
- 逐次入力判定、キー別の苦手記録、苦手を少し優先する出題
- コイン、最小の着せ替え、地域ノードの解放
- ローカル保存、キーボードガイド、動き軽減設定

ゲーム設計の方針は [AI_GUIDE.md](AI_GUIDE.md) と各設計書を参照してください。
