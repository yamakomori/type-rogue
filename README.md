# ことばの小さな冒険 — MVP

子どものタイピング習慣をつくる、短時間のローグライク風タイピングゲームです。Astroのページ層に、ゲーム本体を単一のReact Islandとして載せています。現在はホームポジションから、かなの規則、単語と短いフレーズまでを対象にしたMVPです。

## 起動

```bash
npm run dev
```

ブラウザで表示されたローカルURLを開きます。進捗はブラウザのローカルストレージに保存されます。

## テスト

```bash
npm test
```

## MVPに含むもの

- 潮だまりは3問、浅瀬と珊瑚の森は6問の短いプレイ
- S00〜S08 のホームポジションと上下段、SH01〜SH11 の母音・各行・濁音・促音・撥音・長音・拗音カリキュラム
- CO01〜CO06 の名詞・動作語・様子語・特殊かな混合・助詞つきフレーズ・総合チャレンジ
- 逐次入力判定、キー別の苦手記録、苦手を少し優先する出題
- 潮だまり・浅瀬・珊瑚の森、海域別の水槽と魚、地域ノードの解放
- 5種の4コマ生き物スプライト試験表示（未設定種はCSS魚へフォールバック）
- キー別・ことばの表記別の復習と、導入・練習・混合・宝箱の段階構成
- コイン、最小の着せ替え
- ローカル保存、キーボードガイド、動き軽減設定

ゲーム設計の方針は [AI_GUIDE.md](AI_GUIDE.md) と各設計書を参照してください。
